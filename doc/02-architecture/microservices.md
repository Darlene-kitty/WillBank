# Microservices – Version 1

## 1. Microservice Client
Gestion de :
- création et mise à jour du client  
- informations KYC : nom, prénom, adresse, email, téléphone

## 2. Microservice Compte
Fonctionnalités :
- gestion comptes courants et épargne  
- gestion des statuts et des soldes

## 3. Microservice Transactions
Fonctionnalités :
- virements internes  
- dépôts et retraits  
- paiements marchands  
- historique des transactions

## 4. Microservice Notifications
- email : validation, alertes  
- push mobile (Firebase Cloud Messaging)  
- consommation d’événements RabbitMQ

## 5. Microservice Composite
- tableau de bord client  
- relevé sur une période  
- recherche des transactions
