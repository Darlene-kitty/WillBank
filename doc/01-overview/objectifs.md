# Objectifs du projet

- Concevoir une architecture microservices bancaire
- Définir les frontières de chaque microservice
- Mettre en place une API Gateway et un service Eureka
- Gérer les communications synchrones (REST) et asynchrones (RabbitMQ)
- Implémenter un système de notifications FCM
- Fournir un service composite pour les opérations orientées métier
